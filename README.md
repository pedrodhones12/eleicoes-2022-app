# Sistema de Eleições 2022

Dados das Eleições
A análise do banco de dados das eleições presidenciais de 2022 no Brasil permite compreender, de forma estruturada, o comportamento do eleitorado em diferentes regiões do país. A partir dos dados coletados, é possível observar a distribuição dos votos entre os principais candidatos, com destaque para os partidos Partido dos Trabalhadores (PT) e Partido Liberal (PL).
